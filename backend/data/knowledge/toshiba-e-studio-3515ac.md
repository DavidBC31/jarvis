# Imprimante Toshiba e-Studio 3515ac — Dépannage général

## Codes d'erreur fréquents

Le code E2 indique un bourrage papier dans le bac principal. Ouvrez le capot avant,
retirez délicatement la feuille coincée puis refermez le capot pour réinitialiser.

Le code F5 signale un problème de fusion (unité de fixation surchauffée). Éteignez
l'imprimante, attendez dix minutes qu'elle refroidisse, puis rallumez-la. Si le code
persiste, l'unité de fusion doit être remplacée par le support IT.

Le code C4 correspond à un niveau de toner bas. Remplacez la cartouche de toner noir
par une cartouche neuve référencée T-3515.

## L'imprimante est hors ligne

Vérifiez que le câble réseau est branché et que la LED réseau est allumée. Redémarrez
l'imprimante en la mettant hors tension trente secondes. Si elle reste hors ligne,
contrôlez son adresse IP dans le menu Paramètres réseau et vérifiez qu'elle répond au ping.

## Qualité d'impression dégradée

Des traces ou des bandes sur les impressions indiquent un tambour encrassé. Lancez le
cycle de nettoyage automatique depuis le menu Maintenance, option « Nettoyage tambour ».
