# Accès VPN Bleu Citron

## Première connexion

Installez le client VPN fourni par le support IT, puis saisissez votre identifiant
Bleu Citron et votre mot de passe. Le serveur VPN à utiliser est vpn.bleucitron.lan.

## Impossible de se connecter

Si la connexion échoue, vérifiez d'abord votre accès Internet. Une erreur
« authentication failed » signifie un mot de passe expiré : réinitialisez-le via le
portail interne. Pendant une maintenance planifiée, le VPN peut être temporairement
indisponible — consultez l'état des services sur le dashboard.

## Déconnexions fréquentes

Des coupures répétées proviennent souvent d'un Wi-Fi instable. Privilégiez une connexion
filaire, ou rapprochez-vous de la borne. Réduisez le MTU à 1400 dans les paramètres
avancés du client si le problème persiste.
